Complete mathematical specification of the Circularity State-Space (CSS) Framework for polymer recyclability simulation

🧪 Core State Variables

  • E: Cumulative environmental burden [kg CO₂eq/kg]
  • C: Cumulative net economic cost [USD/kg]
  • Q: Composite quality index [0,1]

📊 Internal State

  • m̄: Normalized molecular weight
  • Xc: Crystallinity fraction
  • Ci: Contamination index
  • π: Purity = 1−Ci

1. The Operating Zone Cuboid

The feasible operating region is defined as a rectangular cuboid in state space:

Z = { s ∈ ℝ³ | E ≤ Emax, C ≤ Cmax, Q ≥ Qmin }
E_max ∈ [5, 80] kg CO₂eq/kg (user-defined)
C_max ∈ [1, 30] USD/kg (user-defined)
Q_min ∈ [0.1, 0.9] (user-defined)

2. Internal State Evolution

Three root-cause variables drive all macroscopic behavior:

sinternal = [m̄, Xc, Ci]
m̄ ∈ [0,1] = M_w(n) / M_w(0) (normalized weight-average molecular weight)
Xc ∈ [0, Xc_max] (crystallinity fraction)
Ci ∈ [0,1] (mass fraction of foreign contaminants)
π = 1 − Ci (purity)

2.1 Use Phase Physics

m̄u = clamp(m̄ · exp(−βu · (1 + Ci)) · (1 + σ · 𝒩(0,1)))
β_u ∈ [0, 0.5] cycle⁻¹ (use-phase chain scission rate)
σ ∈ [0, 0.12] (stochastic noise scale)
Xcu = clamp(Xc + ΔXcG − ΔXcD + σ · 𝒩(0,1))
ΔXc^G = 0.12 · (Xc_max − Xc) (growth toward equilibrium crystallinity)
ΔXc^D = 0.08 · (1 − m̄)² (crack-induced amorphization)
Ciu = clamp(Ci + k · (1 − Ci) · (1 + σ · 𝒩(0,1)))
k ∈ [0, 0.3] cycle⁻¹ (logistic contamination growth rate)

2.2 Recycle Phase Physics

m̄r = clamp(m̄u · exp(−βr) · (1 + σ · 𝒩(0,1)))
β_r ∈ [0, 0.5] cycle⁻¹ (melt processing chain scission)
Xcr = clamp(Xcu · (1 − fm) + Xc0 · fm + kr · (Xcmax − Xcu) + σ · 𝒩(0,1))
f_m ∈ [0,1] (melt memory factor)
k_r ∈ [0.05, 0.15] (recrystallization rate)
Cir = clamp(Ciu · (1 − φ) · (1 + σ · 𝒩(0,1)))
φ ∈ [0,1] (purification efficiency)

2.3 Virgin Blending

m̄b = λ · 1.0 + (1 − λ) · m̄r
Xc_b = λ · Xc₀ + (1 − λ) · Xc_r
Ci_b = (1 − λ) · Ci_r
λ ∈ [0, 0.5] (virgin fraction)

3. Composite Quality Index Q

Q = Qweighted · Qgate
Q_weighted = Σ(w_k · P_k) / Σ(w_k) (weighted average of 8 properties)
Q_gate = Π(P_crit,k / 0.3) (critical property gate)

4. Economic & Environmental Accounting

4.1 Use Phase Costs

ΔEu = Euse + grid · 0.05 · (1 + Ci)
E_use ∈ [0,1] kgCO₂/kg (application-specific)
grid ∈ [0.05,0.9] kgCO₂/kWh
ΔCu = Csort · (1 + Ci)
C_sort ∈ [0,1] USD/kg (collection cost)

4.2 Recycle Phase Costs

ΔEr = ε / max(0.05, Q · π) + λ · Evirgin
ε ∈ [0.1, 5] kgCO₂/kg (technology energy burden)
E_virgin ∈ [0.5, 5] kgCO₂/kg
ΔCr = κ / π + λ · Cvirgin − v0 · Qγ
κ ∈ [0.1, 5] USD/kg (technology cost)
v₀ ∈ [0.1, 2] USD/kg (recyclate price at Q=1)
γ ∈ [0.5, 3] (revenue quality exponent, superlinear)
C_virgin ∈ [0.3, 3] USD/kg

5. Greedy Auto-Selector Algorithm

Cost_Score(θ) = −(ΔCr(θ) + σcarbon · ΔEr(θ))
σ_carbon ∈ [0, 0.3] USD/kgCO₂

Pseudocode:

for each available technology θ:
  Simulate recycle step → s', Q', ΔE, ΔC
  if (Q' ≥ Q_min AND E+ΔE ≤ E_max AND C+ΔC ≤ C_max):
    Select θ with maximum Cost_Score(θ)
  else:
    Select θ with maximum Q' (rescue mode)

6. Recyclability Half-Life τ1/2

τ1/2 = min{ n | Pr[all states s₀...sₙ ∈ Z] ≤ 0.5 }
Computed via Monte Carlo (N=1–20 trajectories)
